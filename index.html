Live Demos

  • Posthog Replay Fixer

    Older Posthog Session Replays point to expired CDN files. Upload an exported replay, update the cdn calls, and export the modified replay. Also view the replay on the site.

  • Chrome Desktop Checker

    Tool for checking whether or not the client is a chrome desktop. Simply checks if the extension runtime is available. Mobile devices do not have access to the 'chrome.runtime'.

  • Keypress Checker

    Tool for checking what buttons were pressed.

  • Paste Debug

    Diagnoses paste issues when pasting into Sourcer. Built to debug Outlook-on-Windows clipboard behavior: inspects clipboard HTML/plain text and character codes (e.g. bullet chars).
